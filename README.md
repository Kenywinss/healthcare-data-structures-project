Healthcare Data Structures Project
Team: Keny's Crew
Member: Keny Nana

How to run:
1. Open a terminal in the folder containing HealthcareDataStructuresApp.java.
2. Compile: javac HealthcareDataStructuresApp.java
3. Run the GUI: java HealthcareDataStructuresApp
4. Run internal tests only: java HealthcareDataStructuresApp test

What the program does:
- Loads sample records or CSV healthcare records.
- Stores PatientRecord objects in a custom dynamic array, linked list, hash map, and priority queue.
- Supports insert, search, delete, traversal, admission processing, and benchmarking.
